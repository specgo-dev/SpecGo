# Contributing

Thanks for considering a contribution. This project uses the Developer Certificate of Origin (DCO).

## DCO Sign-off
By signing off a commit, you agree to the DCO 1.1. Add the following line to your commit message:

```
Signed-off-by: Your Name <your.email@example.com>
```

You can add this automatically with:

```bash
git commit -s
```

## DCO 1.1 (Summary)
You certify that you wrote the code or have the right to contribute it, and that it can be distributed under the project license.

Full text: https://developercertificate.org/
